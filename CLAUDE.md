This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

Professional DJ portfolio website for Javi Beat (javibeat.com). Static site with no build tools, no package manager, and no frameworks — pure HTML5, CSS3, and vanilla JavaScript (ES6+).

## Development

- **Dev server**: VS Code Live Server extension on port 5502
- **Deployment**: GitHub Pages via git push (CNAME: javibeat.com)
- **Server config**: Apache (.htaccess) with security headers, gzip, caching, and hotlink protection
- **No build step, no tests, no linting configured**

## Architecture

**Component loading**: `menu.html` and `footer.html` are loaded dynamically via `fetch()` in `interactions.js` into `#menu-placeholder` and `#footer-placeholder` divs. Every page uses this pattern — changes to navigation or footer only need editing in those two files. After the menu loads, `markActiveLink()` highlights the current page's nav link and `initMenuToggle()` wires the hamburger button. After the footer loads, `reveal()` runs to animate `.reveal` elements.

**Execution flow in `interactions.js`**: DOMContentLoaded → creates custom cursor → loads menu.html + footer.html via fetch → runs `initGlobal()` (month updater, floating WhatsApp button injection) → runs scroll reveal. The WhatsApp button and custom cursor are created entirely in JS, not in any HTML file.

**CSS**: Single `assets/css/styles.css` (plus `404.css` for the error page). Uses CSS custom properties defined in `:root`, mobile-first responsive breakpoints at 480px, 600px, 768px. Design follows "Obsidian Noir" theme — pure black (#000) background, white text, minimal palette. Key custom properties: `--bg-pure`, `--bg-surface`, `--bg-card`, `--accent` (white), `--accent-rebel` (red #e63226), `--font-heading` (Syne), `--font-body` (Space Grotesk), `--font-alt` (Inter).

**JavaScript**:
- `assets/js/interactions.js` — main file: custom cursor (hidden on touch), IntersectionObserver-based `.reveal` scroll animations, component loading, smooth scrolling, floating WhatsApp button
- `assets/js/gallery.js` — lightbox for gallery page
- `assets/js/menu.js` — stub file, all menu logic lives in `interactions.js`

**SEO fragment**: `assets/seo_fragment.html` contains shared meta tags and JSON-LD structured data optimized for AI search engines (ChatGPT, Claude, Gemini). Pages include their own JSON-LD inline rather than loading this fragment.

**External dependencies** (all via CDN, no local packages):
- Google Fonts (Space Grotesk, Syne, Inter)
- Font Awesome 6.0.0
- Google Tag Manager (GTM-N5LD92Z5)

## Pages

`index.html` (homepage/hero), `about.html`, `gigs.html` (performance calendar), `gallery.html`, `djclasses.html`, `press.html`, `404.html`

## SEO & Performance

Heavy JSON-LD structured data and meta tags on each page. Images use WebP format with lazy loading. LCP image is preloaded. CSS for Font Awesome is deferred. Server-side gzip and aggressive caching (1 year for images/fonts, 1 month for CSS/JS, 0 for HTML).

Site content and commit messages are in mixed English/Spanish.
